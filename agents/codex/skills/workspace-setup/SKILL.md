# Workspace Setup

Run from a clone of the workspace-setup repository:

```bash
git clone git@github.com:19tt94/workspace-setup.git && cd workspace-setup && ./run.sh
```

On Windows, use `.\run.ps1` instead of `./run.sh`.

Choose **AI agent starter files** to copy Cursor, Codex, and Claude config without reinstalling apps or devtools.
